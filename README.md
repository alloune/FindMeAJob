# FindMeAJob

Bienvenue sur le projet **FindMeAJob**.

Ce dépôt contient le code source de l'application, divisé en plusieurs parties :

- **backend/** : Le serveur API et la logique métier.
- **frontend/** : L'interface utilisateur web.
- **scraper/** : Les scripts de récupération de données (offres d'emploi).

## Installation

Veuillez vous référer aux instructions spécifiques dans chaque sous-dossier pour installer et lancer les différentes parties de l'application.
